# 画像一括リネームツール

画像ファイルをまとめて連番リネームするツールです。

---

## 必要なもの

```bash
pip install tkinterdnd2 watchdog
```

---

## 起動

```bash
python renamer.py
```

---

## 基本的な使い方

1. **ファイル名**欄にリネーム後のベース名を入力する（例: `vacation`）
2. **桁数**で連番の桁数を選ぶ（例: 3なら `001`, `002`...）
3. 画像ファイルをリストにドラッグ＆ドロップする
   - ドラッグ＆ドロップできない場合は「ファイル追加」ボタンから選択
4. **リネーム実行**を押す
5. **フォルダへ移動**を押すとベース名と同名のフォルダに画像を移動する

リネームは作成日時の古い順に番号が振られます。

---

## フォルダ監視

1. **フォルダ監視: OFF** ボタンを押してフォルダを選ぶ
2. そのフォルダに画像が追加・削除されると自動でリストに反映される
3. もう一度押すと監視を停止する

---

## その他のボタン

| ボタン | 動作 |
|---|---|
| ファイル追加 | ダイアログからファイルを選んで追加 |
| クリア | リストを空にする |
| プレビュー更新 | リネーム後のファイル名を再計算して表示 |
| 最前面: OFF/ON | ウィンドウを常に最前面に固定する |
| 作成日時を表示 | リスト上に作成日時の列を表示する |
